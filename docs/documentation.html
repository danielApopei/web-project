1. Introducere

DeAd (Detention Admin) este o aplicație web pentru gestionarea vizitelor persoanelor condamnate la executarea unei pedepse într-un penitenciar sau într-o casă de corecție.

2.Scop si obiective

Scopul principal al proiectului este să ofere o soluție eficientă pentru gestionarea vizitelor, astfel încât să se asigure o evidență precisă și o comunicare fluidă între condamnați și vizitatori. Obiectivele includ:

  • Înregistrarea datelor relevante despre vizite (identitatea vizitatorului, relația cu condamnatul, natura vizitei etc.).
  • Monitorizarea stării de sănătate și de spirit a detinutului în urma întâlnirilor
  • Generarea de statistici și rapoarte privind vizitele per individ, perioadă de timp și în funcție de categoria pedepsei.

3. Funcționalități principale

  • Înregistrarea vizitelor, inclusiv informații despre vizitatori și natura vizitelor.
  • Consemnarea datelor despre intrevedere, inclusiv data, durata, obiecte/date furnizate, rezumatul discuțiilor.
  • Generarea de statistici privind vizitele, inclusiv documente HTML, CSV și JSON.
  • Oferea funcționalităților esențiale prin intermediul unui API Web REST/GraphQL.

4. Detalii tehnice

Frontend:

Interfața utilizatorului este dezvoltata folosind HTML, CSS și JavaScript. Se asigura o experiență intuitivă și prietenoasă pentru utilizatori.

Tehnologiile folosite pentru acest proiect sunt HTML5, CSS, JavaScript (interfata).

5.3. Pagini si experienta utilizatorului

Welcome page

La aceasta pagina vor avea acces ambele tipuri de utilizatori si administratorii si utilizatorii neconectati. Va fi prima pagina care apare cand cineva intra in aplicatie. De aici, utilizatorul isi va putea programa o vizita sau va putea sa se inregistreze ca administrator

Plan a visit

In aceasta pagina exista un formular care va fi completat de utilizator pentru a isi programa o vizita.

Admin Register

Aceasta pagina se ocupa cu inregistrarea unui nou administrator.

Admin Login

Aceasta pagina se ocupa cu logarea unui administrator existent.

Admin visit info

Aceasta pagina este vizibila doar administratorului. Ea se ocupa cu oferirea de date despre o anumita vizita.

Admin visit list

Aceasta pagina este vizibila doar administratorului. Ea contine o lista cu toate vizitele efectuate.

Admin inmate list

Aceasta pagina este vizibila doar administratorului. Aceasta contine o lista cu toti detinutii din penitenciar sau casa de corectie.

Add new inmate

Aceasta pagina este vizibila doar administratorului. In aceasta exista un formular care, fiind completat de catre administrator, permite adaugarea unui nou detinut in penitenciar, respectiv casa de corectie.

Inmate info

Aceasta pagina este disponibila administratorului pentru a vedea detalii mai amanuntite despre un anumit prizonier.

7.Concluzie

Proiectul DeAd va îmbunătăți procesul de gestionare a vizitelor în penitenciare și va asigura o comunicare eficientă între condamnați și vizitatori. Implementarea acestui sistem va aduce beneficii semnificative în ceea ce privește transparența și eficiența administrării vizitelor.
