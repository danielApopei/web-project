1. Introducere

DeAd (Detention Admin) este o aplicație web pentru gestionarea vizitelor persoanelor condamnate la executarea unei pedepse într-un penitenciar sau într-o casă de corecție.

2. Funcționalități principale

  • Înregistrarea vizitelor, inclusiv informații despre vizitatori și natura vizitelor.
  • Consemnarea datelor despre intrevedere, inclusiv data, durata, obiecte/date furnizate, rezumatul discuțiilor.
  • Generarea de statistici privind vizitele, inclusiv documente HTML, CSV și JSON.
  • Oferea funcționalităților esențiale prin intermediul unui API Web REST/GraphQL.

3. Structura de bază a paginii

Fiecare pagină HTML a aplicației are o structură de bază comună:

<!DOCTYPE html>

<head>
<!-- Metadate și legături către fișierele CSS și JavaScript -->
